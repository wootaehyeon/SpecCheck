# 🖥️ SpecCheck-AI: AI 기반 PC 견적 검증 시스템

[cite_start]**"검증"** — 사용자가 직접 구성한 PC 견적의 가격, 성능, 호환성을 AI가 객관적으로 평가합니다[cite: 4, 9].

---

### 📌 Project Summary
* [cite_start]**목표**: 사용자가 입력한 견적의 가격 적정성, 성능 균형, 부품 호환성 및 용도 적합성을 종합 분석하여 평가 결과를 제공하는 AI 시스템 구축[cite: 9].
* [cite_start]**차별점**: 기존의 추천 중심 서비스와 달리, 사용자가 이미 선택한 견적의 품질을 객관적으로 검증하고 분석 결과에 대한 이유를 함께 제공[cite: 11, 22, 23].

### ✨ Key Features
* [cite_start]**가격 검증**: 실시간 시세와 사용자 입력 가격을 비교하여 부품별 가격 적정성 및 오버페이 여부 판단[cite: 7, 14].
* [cite_start]**성능 밸런스 분석**: 성능 점수 기반의 병목(Bottleneck) 현상 탐지 및 비효율적인 조합 분석[cite: 7, 14].
* [cite_start]**호환성 검증**: CPU-메인보드, RAM, 파워 등 부품 간 조립 가능 여부 자동 확인[cite: 7, 14].
* [cite_start]**용도 적합성 평가**: 게임, 개발, 영상편집, AI 등 사용 목적에 따른 성능 대비 적합도 분석[cite: 7, 14].
* [cite_start]**구매 리스크 점수**: 가격, 성능, 호환성을 종합하여 구매 위험도 점수 산출 및 개선 방향 제시[cite: 7, 8, 14, 15].
* [cite_start]**LLM 기반 설명**: 분석 결과를 자연어 형태로 제공하여 비전문가도 쉽게 이해할 수 있도록 지원[cite: 8, 15, 25, 36].

### ⚙️ Operating Concept
1. [cite_start]**입력**: 사용자가 웹을 통해 PC 견적(부품, 가격, 용도) 입력[cite: 15, 26].
2. [cite_start]**조회**: 시스템이 입력된 견적 기반의 가격, 성능, 호환성 데이터 조회[cite: 16, 27].
3. [cite_start]**분석**: ML 모듈이 가격 적정성, 성능 균형 등을 정량적으로 평가하여 위험도 점수 산출[cite: 17, 18, 28, 29].
4. [cite_start]**생성**: LLM 모듈이 분석 결과를 바탕으로 사용자 친화적인 설명 및 개선안 생성[cite: 18, 29].
5. [cite_start]**결과**: 사용자가 자신의 견적에 대한 종합 평가 리포트 확인[cite: 19, 30].
