# PC 부품 3D 조립 시뮬레이터 - No WebGL 버전

이 버전은 WebGL을 사용하지 않고, HTML5 Canvas 2D로 3D 좌표를 직접 투영해서 렌더링합니다.
따라서 Chrome에서 "WebGL을 사용할 수 없음" 메시지가 떠도 실행됩니다.

## 실행 방법
1. 압축을 풉니다.
2. `index.html`을 Chrome 또는 Edge로 더블클릭해서 엽니다.

## 조작법
- 마우스 드래그: 시점 회전
- 휠: 확대 / 축소
- 부품 클릭: 선택
- 빛나는 슬롯 클릭: 자동 장착
- 장착된 부품 선택 후 [선택 부품 장착 해제]

## 조립 규칙
- 메인보드를 먼저 장착해야 CPU/RAM/SSD/GPU 장착 가능
- CPU를 장착해야 쿨러 장착 가능
- PSU는 바로 장착 가능
